    SETTING YOUR DATABASE

    1. open phpmyadmin panel
    2. click on new 
    3. Import the "gst_db_template.sql" file from the current directory


    SETTING THE PROJECT

    1. create a folder named "gst";
    2. extract or move these file into created folder
    3. move the gst folder to htdocs folder of xamp stack;
    4. Under root directory open config.php 
    5. fill the credential of your database, expected from line 4 to 7;
    6. run the apache, run the project under http://localhost/gst
